# AstroWizard website

The download/landing page for [AstroWizard](https://astrowizard.lukomatico.com/) — a beginner-friendly astrophotography processing app by [Lukomatico](https://youtube.com/@lukomatico).

This repository contains only the website. The application itself is developed separately.

- `index.html` — the whole site (self-contained, no build step)
- `version.json` — machine-readable pointer to the latest build (used by the app's update check)
- `CNAME` — custom domain for GitHub Pages
